    Windows Subsystem for Linux

    WSL transforms your Windows environment with Linux capabilities.

    wsl --install :installs linux on your computer wsl : opens Linux in your terminal clear : clears the terminal screen logout or exit : returns to command prompt in Windows
    Apache Commands

    Manage your web servers settings via these Apache Terminal commands

    sudo apt update : updates your Linux before installation of Apache sudo apt upgrade : upgrades linux sudo service apache2 start : starts the Apache server
    Apache Setup

    Configure your apache settings for your projects.

    sudo nano : open text editor sudo chown -R username path/to/working/directory : make folder writable sudo nano /etc/apache2/sites-available/000-default.conf

    Enable site; look for document root and change it to your desired folder to run in localhost.

    Folder Locations
    • Apache Folder - etc/apache2
    • Document Root - /var/www/
